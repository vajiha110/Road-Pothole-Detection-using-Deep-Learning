# Road Pothole Detection using Deep Learning

A deep learning-based web application that detects road potholes from images using TensorFlow, Flask, and OpenCV. The project helps identify damaged road surfaces automatically through image processing and machine learning techniques.

---

## 🚀 Features

- Upload road images for pothole detection
- Deep learning-based image classification
- Flask web interface
- Image preprocessing using OpenCV
- Real-time prediction display
- Simple and user-friendly UI

---

## 🛠️ Technologies Used

- Python
- Flask
- TensorFlow / Keras
- OpenCV
- NumPy
- HTML/CSS

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/Road-Pothole-Detection-using-Deep-Learning.git
```

### 2. Navigate to Project Folder

```bash
cd Road-Pothole-Detection-using-Deep-Learning
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python app.py
```

Open browser and visit:

```bash
http://127.0.0.1:5000
```

## 🧠 How It Works

1. User uploads a road image
2. Image is preprocessed using OpenCV
3. Deep learning model analyzes the image
4. Prediction result is displayed on the webpage
